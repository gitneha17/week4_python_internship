## 📌 Overview
This project focuses on analyzing sales data and creating meaningful visualizations using Python.  
The goal is to represent data insights clearly using different types of charts and graphs.

## 🎯 Objectives
- Understand data visualization techniques  
- Work with real-world dataset (sales data)  
- Generate different types of plots for better insights  
- Improve data analysis and presentation skills  

## 🛠️ Tools & Technologies Used
- Python  
- Pandas  
- Matplotlib  
- Seaborn

## 📊 Visualizations Created

### 🔹 Bar Chart
- Represents product-wise sales comparison  

### 🔹 Histogram
- Shows distribution of sales values  

### 🔹 Scatter Plot
- Displays relationship between sales and quantity  

### 🔹 Heatmap
- Shows correlation between numerical variables  

## 📈 Key Insights
- Electronics category shows higher sales compared to others  
- Some products have high sales but low quantity (high-value items)  
- Strong relationships observed between certain variables
